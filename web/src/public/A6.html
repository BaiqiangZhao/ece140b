TheraBot

ECE140b Assignment 6: Product ID

Part 1 - Features vs Benefits

1. Microphone Feature

TheraBot is able to collect audio from users so that it is listening to our users and able to react with what it collected.

2. Speaker Feature

TheraBot can talk to users by a speaker. Users can get audio responses when they talk to TheraBot. By this feature, TheraBot makes Users feel it is interactive.

3. Display Screen Feature

TheraBot can express its emotion by showing different facial expression on its display. It makes users think that TheraBot has charactors and it is fun to interact with TheraBot.

4. App Configuration Feature

Users can configurate TheraBot on App which makes TheraBot more customized and more users would like it.

5. Additional Functions

TheraBot has additional functions like alarm clock, reminders which make TheraBot more helpful and show users the attitude of being helpful.

Part 2 - product interactions

The key product interactions are being a companion of users by listening and responding what users talk to it and building a emotional bonding with users to relieve users lonliness and stress.
